文件上传

基于angular-file-upload，进行改造，筛选相关功能，修改相关的样式，并且封装成ucFileUpload指令。
1.引入angular-file-upload,npm install angular-file-upload;
2.更多API请查看angular-file-upload官方文档。
如何使用
1.引入ucFileUpload指令：<uc-file-upload upload="uploader()" result="upload" url="upload.do"></uc-file-upload>;
2.url string 上传接口;
3.upload func 回调函数;
4.result string 上传完成后，后台的返回数据。
