金矿镇（Gold Mine Town）。
```
在200多年前美国西部的一个小山谷里，终日做着发财梦的淘金者开矿镇山、建市镇、设赌场，于是这里成了金矿小镇。但天有不测风云，那里发生了百年不遇的山洪，于是人们便在汹涌的洪水中四处逃命…… 　　
重点项目：矿山车、金矿漂流、魔幻剧场、矿井戏水、淘金河、金矿听音室、香格里拉森林
```

飓风湾（Typhoon Bay）。
```
一个充满浓郁加勒比海岛风情的废墟小镇上，飓风和地震侵袭的斑驳残迹隐约可辨，满目疮痍，奇遇无穷。 　　
重点项目：激流勇进、完美风暴、飓风眼、星球大战、反斗乐。
```

玛雅水公园（Playa Maya Water Park）。
```
每年5月1日——10月7日免费开放！亚洲唯一荣获世界水公园协会“行业创新”桂冠的水公园。原汁原味南美风情，各种精妙嬉水设施，让您领略玛雅文明的神奇，享受嬉水的乐趣！
```

```
过山车是欢乐谷中最刺激的项目了。
```

节目表演：
```
地道战
　　影视拍摄场上演的大型影视实景拍摄表演《地道战》，生动地再现了当年抗日战争波澜壮阔的历史画卷。亲身置身其中，更能体现到影视拍摄的欢乐。
极限运动
　　特邀国外极限运动高手进行现场表演，精彩动作将令您血液沸腾！

高空杂技
　　技术精湛的专业杂技演员，将为游客表演精彩绝伦的高空杂技。
街舞
　　最具国际潮流的街舞表演，劲爆的音乐，强烈的节奏，怦然心动的街舞，将让您也动起来。
炫彩巡游
　　中国首创夜光大巡游！流光溢彩，绚丽缤纷，引爆都市夜生活高潮！另人陶醉的动感乐曲、盛大的游行、眩目的色彩、激越的舞步，燃起在场所有游客的激情！点亮所有人的欢乐夜生活。

游玩季节：由于深圳没有冬天。夏天时间居长。所以全年都可以去欢乐游玩。只是天气原因，比如下雨，气温较低的时候可能会影响到游玩。但深圳大多数天气还是睛天为主的，所以很多时间都是可以去游玩的。
```
