## 欢乐谷

12点半看：
```
四维电影
UfO
发现者
尖峰时刻
（没有雪域雄鹰和雪山飞龙刺激，但也还不错）
```
在去：
```
追风者
彩云之翼
```

然后在去：
```
激流勇进
金矿淘流
（ 需要带雨衣，10元一件，园内禁止自带雨衣）

雅玛水公园，是个很大的区域，可以玩很久
```

不需要玩的项目：
```
寻梦河
异度空间
魔幻城堡
还园小火车
```
